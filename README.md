# Trinity

Trinity is a monorepo containing a mobile shopping application and its supporting backend infrastructure. It is split into three main parts:

- **API** — A REST API built with Node.js, deployed on Vercel, using Drizzle ORM and PostgreSQL for data persistence.
- **Mobile** — A cross-platform mobile app built with Expo and React Native, featuring product browsing, a cart, order history, and deal recommendations.
- **Seed** — A utility package for seeding the database with initial data.
